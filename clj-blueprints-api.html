API for clj-blueprints - clj-blueprints ()


Usage: 
(ns your-namespace
  (:require clj-blueprints))

Overview


Wrapper for the TinkerPop Blueprints API for Graph DBMSs. It supports version 0.7 of the Blueprints API.
